# Atividade 5 - Autômatos Finitos

## Descrição
Criamos um DFA para reconhecer números inteiros.

## Explicação
Criamos um autômato finito determinístico (DFA) para reconhecer números inteiros.
O autômato possui estados e transições que representam dígitos.
Isso demonstra como scanners utilizam autômatos para reconhecer padrões.
